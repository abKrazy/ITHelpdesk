# Outlook Email Troubleshooting

## Overview
Assists users experiencing Outlook email issues.

## Resolution Steps
1. Restart Outlook.
2. Verify internet access.
3. Test Outlook Web Access.
4. Refresh shared mailboxes.
5. Rebuild search index.

## When to Create a Ticket
- Email unavailable for extended period
- Multiple users affected
- Permission issues persist

## Recommended Assignment Group
M365 Support
