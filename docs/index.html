ServiceNow IT Helpdesk AI Agent on Azure

A production-style multi-agent accelerator: end users chat in a CopilotKit / Next.js UI, an AG-UI backend invokes a Foundry-hosted Orchestrator that routes to specialist agents to resolve from a knowledge base or create, assign, check & update ServiceNow incidents — with human-in-the-loop approval on every write.
