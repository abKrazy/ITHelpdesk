# Software Installation Request

## Overview
Explains how users request approved software.

## Symptoms
- Required software missing
- Installation denied
- Application unavailable in portal

## Resolution Steps
1. Search company software portal.
2. Verify application approval status.
3. Submit request with business justification.
4. Await approval workflow if required.

## When to Create a Ticket
- Application unavailable
- Installation repeatedly fails
- Approval required

## Recommended Assignment Group
Endpoint Management

## Keywords
software install, application request, licensing, software center
