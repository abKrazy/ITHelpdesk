# Laptop Running Slow

## Overview
Helps users resolve common system performance issues.

## Symptoms
- Slow application startup
- Computer freezing
- Delayed response
- High CPU or memory usage

## Resolution Steps
1. Restart computer.
2. Check disk space.
3. Close unnecessary applications.
4. Install pending updates.
5. Review Task Manager.

## When to Create a Ticket
- Performance remains poor after reboot
- CPU remains above 90%
- Hardware issues suspected

## Recommended Assignment Group
Desktop Support

## Keywords
slow laptop, performance issue, high CPU, memory issue
