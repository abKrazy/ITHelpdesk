# Unable to Login to Corporate Applications

## Overview
Addresses users unable to sign in despite having valid credentials.

## Symptoms
- Login screen loops
- Authentication fails
- Blank page after login
- SSO does not complete
- MFA challenge never appears

## Resolution Steps
1. Verify internet connectivity.
2. Open an incognito/private browser session.
3. Clear browser cache and cookies.
4. Verify MFA enrollment and connectivity.
5. Restart the device.

## When to Create a Ticket
- Authentication still fails after troubleshooting
- MFA registration is broken
- User account disabled
- Multiple users impacted

## Recommended Assignment Group
Identity and Access Management

## Keywords
cannot login, authentication failure, SSO, MFA, login loop
