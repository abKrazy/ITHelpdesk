# VPN Connectivity Troubleshooting

## Overview
Assists remote workers who cannot connect to the corporate VPN.

## Symptoms
- VPN not connecting
- Connection timeout
- VPN disconnects repeatedly
- Unable to access internal applications

## Resolution Steps
1. Verify internet access.
2. Disconnect and reconnect VPN.
3. Verify credentials.
4. Restart VPN client and device.
5. Update VPN software.
6. Test from an alternate network.

## When to Create a Ticket
- VPN still fails after troubleshooting
- Multiple users experience the issue
- Error code persists

## Recommended Assignment Group
Network Support

## Keywords
VPN, remote access, connection timeout, network issue
