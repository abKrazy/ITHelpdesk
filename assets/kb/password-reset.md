# Password Reset and Login Assistance

## Overview
This article helps users who are unable to sign in, forgot their password, or are receiving authentication errors.

## Symptoms
- Forgot password
- Invalid username or password message
- Password expired
- Account locked after failed attempts
- Repeated login prompts
- Unable to access company applications due to authentication failure

## Common Causes
- Expired password
- Incorrect password entered multiple times
- Account temporarily locked
- Password cached incorrectly on device
- User recently changed password but applications are still using old credentials

## Resolution Steps
1. Verify username.
2. Use self-service password reset.
3. Restart affected applications.
4. Reauthenticate and complete MFA.

## When to Create a Ticket
- Self-service reset fails
- Account remains locked
- MFA device unavailable
- Authentication methods inaccessible

## Recommended Assignment Group
Service Desk

## Keywords
password reset, forgot password, account locked, login issue, MFA, authentication
